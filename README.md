# Smart-Desktop-Assistant

# 🤖 Python Voice Assistant (Basic)

This is a simple voice assistant built using Python.
It can perform basic tasks like opening websites, telling time, and launching system applications.

## 🚀 Features

* 🕒 Tell current time, date, and day
* 🌐 Open websites like YouTube, Google, Instagram
* 🗺 Open Google Maps
* 💻 Open system apps (Notepad, Calculator)
* 🗣 Text-to-Speech response

## 🛠 Technologies Used

* Python
* pyttsx3
* webbrowser
* datetime
* os

## ▶️ How to Run

1. Install Python
2. Install required libraries:

   ```bash
   pip install pyttsx3
   ```
3. Run the file:

   ```bash
   python assistant.py
   ```
